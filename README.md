## 👋 About Me

Hi, I’m **Juyong Jeong**, a M.S./Ph.D. integrated student in **Computer Science** at  
**Mixed Reality Lab., Hanyang University**.

I work around the intersection of **Computer Vision**, **Computer Graphics**, and **XR systems**, with a focus on building visual technologies that connect real-world spaces and virtual environments.

Currently interested in:

- **XR / VR / AR**
- **Game Graphics**
- **Visual Positioning System**
- **3D Scene Understanding**
- **Real-to-Virtual Environment Alignment**
